# Proyecto PizzaTrack

Sistema de gestión de pedidos de una pizzería desarrollado en Java.

## Funcionalidades
- Registrar pizza
- Deshacer pedido (Undo)
- Rehacer pedido (Redo)
- Mostrar pedido actual
